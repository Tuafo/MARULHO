# Semantics

This package contains grounded concept stores, diagnostics, cognitive-signal
records, language result/packet data structures, curiosity, and provenance.

It does not generate language. The MARULHO Transformer is owned by
`training`, installed by `MarulhoBrain`, and evaluated by `evaluation`.

The retired spike-language decoder, SNN readout surface, and service-owned
language plasticity contracts were removed. Future grounding work should connect
measured object/action/intervention representations to the active Transformer
through a small, explicit experiment rather than revive the old readout stack.
