# Interaction

Use this with [../../../README.md](../../../README.md) and
[../../../CONTEXT.md](../../../CONTEXT.md).

`interaction` owns operator-facing response shaping over evidence surfaces.

## Owns

- Grounded response formation.
- Evidence selection and presentation for operator interaction.

## Must Not Own

- Hidden ThoughtLoop behavior.
- Unsupported fluency claims.
- Fact/action promotion without the relevant evidence gates.

## Runtime Rules

Interactions should expose what MARULHO evidence supports and what remains
uncertain. Do not turn a readable response into proof of cognition, grounding,
or autonomous action.
